# biblebot
BibleBot for Stoat Chat
## Join [Faith Christian Community Server](https://stt.gg/3vTTwtM8) in Stoat Chat

## Features : 
1. !random - random verse generator.
2. !daily - return daily verse using beta.ourmanna.com API. 
3. !John 3:16 - return full verse.
4. !John 3:16-18 - return verse range.
5. !John 3:16, !Mark 1:1, !Luke 6:37-38 - return multiple verses & verse range.
6. John 3:16 KJV - return full verse in KJV version, John 3:16 ASV - return in ASV version.
7. !setversion [BIBLE_VERSION] - set default version according to user.
8. !versions - list all support Bible versions.
> Versions: WEB, KJV, ASV, BBE, DARBY, DRA, YLT, OEB-US, OEB-CW, WEBBE, ALMEIDA, RCCV, BKR, CUV, CLEMENTINE, CHEROKEE
9. !pingbot - return status & default version.
10. Simple banned words filter - Auto moderator bot function, delete message when banned words detected.
